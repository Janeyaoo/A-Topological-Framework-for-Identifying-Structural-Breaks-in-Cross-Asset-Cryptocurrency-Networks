# Replication Materials: A Topological Framework for Identifying Structural Breaks in Cross-Asset Cryptocurrency Networks

**Manuscript ID:** CSEM-D-25-03144  
**Journal:** Computational Economics  
**Authors:** Junda Wu, Jian Yao, Haoyuan Feng

---

## Overview

This repository contains all data, code, and figures necessary to replicate the main empirical results of the paper. The analysis builds a TDA-based framework that tracks the topological evolution of cross-asset cryptocurrency similarity networks to detect structural breaks and regime shifts.

The full pipeline proceeds in seven sequential steps, each corresponding to a numbered notebook.

## Data Description

### Raw Input Data (`data/raw/`)

| File | Description | Source | Period |
|------|-------------|--------|--------|
| `Close_price_10_raw.xlsx` | Daily closing prices for top-10 cryptocurrencies by market cap (BTC, ETH, XRP, USDT, BNB, SOL, USDC, DOGE, TRX, ADA) | CoinMarketCap | Aug 2020–Sep 2025 |
| `Volatility_10_raw.xlsx` | High-low price range data for volatility estimation, top-10 cryptos | CoinMarketCap | Aug 2020–Sep 2025 |
| `Parkinson_Rogers_Volatility.xlsx` | Parkinson and Rogers-Satchell volatility estimates | Computed from OHLC | Aug 2020–Sep 2025 |
| `SP500_VIX_benchmarks_raw.xlsx` | S&P 500 index and CBOE VIX daily series | Yahoo Finance | Aug 2020–Sep 2025 |
| `Fear_Greed_Index_raw.xlsx` | Crypto Fear & Greed Index (daily) | alternative.me | Aug 2020–Sep 2025 |
| `fear_and_greed_index.csv` | Crypto Fear & Greed Index (processed, CSV format) | alternative.me | Aug 2020–Sep 2025 |
| `Event_Catalog.xlsx` | 37 catalogued cryptocurrency market events with dates, categories, and descriptions | Authors' compilation | Aug 2020–Sep 2025 |

### Key Processed Dataset

The main analysis dataset `TDA_SP500_VIX_BTC_BVOL_CVI_merged0329_labeled_with_network.csv` merges all TDA features (Betti-0, Betti-1, persistent entropy for log-return and volatility under DTW and Wasserstein distances), benchmark market indicators (VIX, BVOL, S&P 500, Fear & Greed, BTC Rogers volatility), event labels (37 events, 8 categories), and network centrality metrics into a single panel. This file is the direct input for steps 4–8 and can reproduce all main paper results without re-running steps 1–3.

## Replication Workflow

Run notebooks in order:

1. **step1** — Loads raw closing prices and volatility series, computes log-returns, and constructs sliding-window delay embeddings (baseline: m=21, τ=1).

2. **step2** — Computes pairwise Wasserstein (p=1, via `scipy.stats.wasserstein_distance`) and DTW (via `fastdtw`) distance matrices for each rolling window date. Output: time-indexed 10×10 distance matrices.

3. **step3** — Applies Vietoris–Rips filtration (via `ripser`, maxdim=1) to each distance matrix. Extracts Betti-0, Betti-1, and persistent entropy evaluated on 50 equally spaced filtration values up to the global maximum pairwise distance (DTW: 5.321886, Wasserstein: 0.427174).

4. **step4** — Performs ROC-AUC benchmarking of all TDA indicators against VIX, BVOL, S&P 500, Fear & Greed, and BTC Rogers volatility, using a ±21-day event window around 37 catalogued events. Includes bootstrapped 95% CIs and DeLong pairwise tests.

5. **step5** — Category-stratified AUC analysis across 8 event types (Regulation, Systemic Failure, Market Milestone, Institutional Adoption, Corporate Action, Protocol/Tech, Macro Shock, Security/Hack).

6. **step6** — Case studies for four representative events: BTC Bull Market Peak (2021-04-10), China Crypto Ban (2021-09-24), Terra/LUNA Collapse (2022-05-01), FTX Collapse (2022-11-01).

7. **step7** — Granger causality tests (lags h ∈ {1,3,5,7,10,14,21}) between all four Betti-0 indicators and five market stress measures. Includes ADF unit-root pre-testing.

8. **step8** — Topological event study: z-scored Betti-0 trajectories around events, aggregated by category.

## Key Implementation Parameters

| Parameter | Value | Notes |
|-----------|-------|-------|
| Sliding window length m | 21 (baseline) | ≈ one trading month |
| Time lag τ | 1 | Rapid ACF decay in crypto returns |
| Distance metric | Wasserstein (p=1) + DTW | Both applied to log-return and volatility series |
| VR maximum dimension | 1 | H₀ and H₁ only |
| Filtration grid points | 50 | Equally spaced from 0 to global max distance |
| Max DTW distance | 5.321886 | Global max across all rolling windows |
| Max Wasserstein distance | 0.427174 | Global max across all rolling windows |
| Event window (baseline) | ±21 days | Also tested: ±7, ±14 days |
| TDA library | Ripser.py (ripser) | Bauer (2021) |
| Change-point library | ruptures | PELT (RBF, penalty=log(n)), BinSeg (RBF, n_bkps=10), KernelCPD (linear, n_bkps=10) |
